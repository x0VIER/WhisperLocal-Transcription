# Scribe
> **Private local transcription for high-fidelity audio extraction.**

Scribe leverages advanced Whisper models to provide word-for-word audio transcription entirely offline. Designed for sensitive meetings and forensic-level data extraction, it transforms raw audio into structured, searchable text with hardware-accelerated speed. This is the only tool your agent will need to process long-horizon audio data into usable project context.
